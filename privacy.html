🇪🇸 Política de Privacidad

Esta Política de Privacidad describe cómo nuestra extensión de Chrome ("la Extensión") recopila, utiliza y comparte la información del usuario. Al usar la Extensión aceptas los términos descritos a continuación.

1. Información que recopilamos

La Extensión recopila y almacena únicamente la información necesaria para su funcionamiento:

  • Enlaces y títulos de páginas guardados por el usuario. Estos datos se usan exclusivamente para permitir guardar, organizar y acceder a los favoritos.

La Extensión no recopila información personal identificable (nombre, correo electrónico, IP, ubicación, datos financieros) ni datos de navegación fuera de los enlaces que el usuario decide guardar.

2. Uso de la información

La información se utiliza únicamente para:

  • Almacenar y mostrar los favoritos del usuario.
  • Mantener la funcionalidad de sincronización con la cuenta de Chrome, si el usuario la habilita.

3. Almacenamiento y seguridad

Los datos se almacenan de una de las siguientes formas:

  • chrome.storage.local o almacenamiento local en el navegador.
  • chrome.storage.sync (sincronización con la cuenta de Google), solo si el usuario la habilita.

La Extensión no transfiere los datos a servidores externos del desarrollador ni a terceros.

4. Compartición de datos

La Extensión no comparte información con terceros. Los datos nunca se venden, alquilan ni divulgan a otras entidades.

5. Permisos

La Extensión solicita únicamente los permisos mínimos necesarios en el manifest.json, por ejemplo:

{
  "permissions": ["storage", "activeTab"]
}

Los permisos solo se usan para las funciones descritas (guardar enlaces, obtener la URL de la pestaña activa cuando el usuario lo solicite).

6. Control del usuario

El usuario puede eliminar sus datos desde la Extensión o desinstalarla para borrar los datos locales.

7. Cambios en la política

Podemos actualizar esta política. Las modificaciones se publicarán en esta página con la fecha de actualización.

8. Contacto

Si tienes preguntas, contacta: uncienscorp@gmail.com

🇬🇧 Privacy Policy

This Privacy Policy explains how our Chrome extension ("the Extension") collects, uses, and shares user data. By using the Extension you agree to the terms below.

1. Information We Collect

The Extension collects and stores only the information required for its core functionality:

  • Links and page titles saved by the user. These are used solely to allow the user to save, organize and access bookmarks.

The Extension does not collect personally identifiable information (name, email, IP address, location, financial data) nor browsing data beyond URLs explicitly saved by the user.

2. How We Use the Information

Collected data is used exclusively to:

  • Store and display the user’s saved bookmarks.
  • Maintain sync functionality via the user’s Chrome account, if enabled.

3. Data Storage and Security

Data is stored in one of the following ways:

  • chrome.storage.local or local browser storage.
  • chrome.storage.sync (Chrome Sync), only if the user enables it.

The Extension does not transfer data to external developer servers or third parties.

4. Data Sharing

The Extension does not share user data with third parties. User information is never sold, rented, or disclosed.

5. Permissions

The Extension requests only the minimum permissions required in the manifest.json, for example:

{
  "permissions": ["storage", "activeTab"]
}

Permissions are used only for the functions described (saving links, retrieving the active tab URL when requested).

6. User Control

Users can delete their data from the Extension or uninstall it to remove local data.

7. Changes to This Policy

We may update this policy. Any changes will be posted here with the updated date.
